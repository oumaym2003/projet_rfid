# pfe-image-collection Frontend Documentation

## Overview
This project is designed to assist medical professionals in uploading and categorizing medical images. The frontend is built using Tkinter, providing a user-friendly interface for image insertion and disease classification.

## Features
- Image upload functionality
- Selection of disease types from a dropdown menu
- Checkbox options for disease classification
- User-friendly interface tailored for medical professionals

## Project Structure
The frontend consists of the following key files:
- `main.py`: The entry point for the Tkinter application.
- `gui/tkinter_interface.py`: Contains the main GUI layout and functionality.
- `gui/helpers.py`: Includes helper functions for image processing and validation.

## Installation
To set up the frontend, ensure you have Python installed. You can then run the application using the following command:

```bash
python main.py
```

## Usage
1. Launch the application.
2. Use the interface to upload images.
3. Select the appropriate disease type from the dropdown menu.
4. Check the relevant disease classification options.
5. Submit the form to save the image and its associated metadata.

## Future Work
This project aims to integrate with a Django backend for data storage and processing. Future updates will include enhanced image processing capabilities and improved user experience based on feedback from medical professionals.
